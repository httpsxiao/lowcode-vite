# lowcode

基于：

- vite
- vue-next composition api
- vue-router-next
- vuex-next
- koa

搭建的建站平台，仅用于学习

```
|-- mock 开发时 server
|-- src 源码
    |-- asserts 资源
    |-- components 网站组件
    |-- library 物料库
    |-- plugins 基于 vue-next 的插件
    |-- router 路由
    |-- store 状态管理
    |-- template 生成页面模板
    |-- utils 工具库
```
